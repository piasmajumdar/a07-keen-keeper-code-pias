# Keen Keeper

## Project Description

**Keen Keeper** is a smart friend-tracking application that helps you maintain meaningful relationships. It allows you to log interactions like calls, texts, and video chats, track when you last connected with someone, and stay consistent with your communication goals.

The app provides a clear overview of your friendships, detailed interaction timelines, and visual analytics to understand your communication patterns better.

---

## Technologies Used

* React
* npm
* Tailwind CSS
* DaisyUI
* JavaScript (ES6+)
* Recharts
* React Icons
* Google Fonts
* CSS3

---

## Key Features

### 1. Friend Management Dashboard

* View all your friends on the homepage
* Click on a friend to access detailed interaction history
* Track last contact date and interaction goals

### 2.  Interaction Timeline

* Log interactions like **Call, Text, Video Call**
* Automatically record date and time of each interaction
* View a complete timeline of all activities

### 3. Analytics & Insights

* Visualize interaction data using **Pie Charts**
* See breakdown of communication types (calls, texts, video)
* Get insights into your social engagement patterns
